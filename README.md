# QtNationalDex

This is an update of the NationalDex project to include images to make the program better. This program is capable of producing data for from Kanto to Alola. The following is an example of what the program would look like if the user picked Gengar

The information and images are © Nintendo/Creatures Inc./GAME FREAK Inc. I do not own anything used in this program, just a fan of Pokemon

# To Use

This project was created using [Qt](https://www.qt.io/). Download the project and open it up in Qt to run the application or open up the release folder and run "NationalDex.exe".
